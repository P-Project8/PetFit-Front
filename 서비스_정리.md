# PetFit 서비스 정리

> AI 기반 가상 피팅(Virtual Try-On) 중심의 반려견 의류 쇼핑 플랫폼
> 최종결과보고서 작성용 서비스 개요 문서

---

## 1. 서비스 개요

| 항목 | 내용 |
|------|------|
| 서비스명 | PetFit |
| 한 줄 정의 | AI 가상 피팅 기능을 핵심으로 하는 반려견 의류 쇼핑 플랫폼 |
| 핵심 가치 | 반려견 사진 + 옷 선택 → AI 합성으로 착용 모습 미리 확인 → 구매까지 연결 |

## 2. 핵심 기능 — AI 스타일링 (가상 피팅)

서비스의 메인 기능. 경로 `/ai-styling` (로그인 필수)

### 사용 흐름

1. **반려견 사진 준비** — 직접 업로드 또는 등록된 '내 반려견 프로필'에서 선택
2. **옷 선택** — 상품 목록에서 선택, 상품 상세페이지의 'AI 스타일링' 버튼으로 진입, 또는 옷 이미지 직접 업로드
3. **AI 합성** — 반려견 사진 + 옷 이미지를 서버로 전송, AI가 착용 모습 합성 (로딩 스플래시 화면 노출)
4. **결과 활용** — 결과 이미지 저장 / 갤러리 공유 / 유사 상품 추천

### 세부 특징

- 최초 진입 시 사용법 안내 온보딩 모달 노출 (1회 노출 후 미표시)
- 합성 결과 생성 후 선택한 옷과 동일 카테고리의 유사 상품 자동 추천 → 구매 연결
- 합성 요청 시 선택한 반려견 프로필·상품 정보를 함께 전달하여 사이즈/추천 기능과 연동

## 3. 회원 / 인증

| 기능 | 설명 |
|------|------|
| 이메일 인증 회원가입 | 인증코드 발송 → 코드 검증 → 회원정보(이름·생년월일 등) 입력 → 가입 완료 |
| 로그인 / 로그아웃 | 아이디·비밀번호 기반 인증 |
| 자동 로그인 유지 | accessToken / refreshToken 브라우저 저장 |
| 토큰 자동 재발급 | 토큰 만료(401) 시 자동 재발급으로 끊김 없는 이용, 재발급 실패 시 자동 로그아웃 |
| 프로필 조회 / 수정 | 이름·생년월일 등 회원정보 관리 |
| 로그인 보호 라우트 | 장바구니·결제·찜·AI스타일링·마이페이지는 비로그인 시 접근 차단 |

## 4. 반려견 프로필 관리

내 반려견 정보를 등록·관리. AI 스타일링 및 사이즈 추천에 활용.

| 기능 | 설명 |
|------|------|
| 등록 / 수정 / 삭제 | 이름, 견종, 나이, 몸무게, 신체 치수(목둘레·가슴둘레·등길이), 사진 |
| 멀티 펫 | 여러 마리 등록 가능 |
| 사이즈 추천 | 등록 치수 기반으로 특정 상품의 추천 사이즈와 근거 제공, 옵션별 핏(딱맞음/넉넉함 등) 설명 |
| 비슷한 체형 큐레이션 | 유사 체형 견주들이 많이 구매한 인기 상품 추천 (구매 수·인기 퍼센트 표시) |

## 5. 쇼핑 (이커머스)

### 상품 탐색

| 기능 | 설명 |
|------|------|
| 홈 화면 | 자동 슬라이드 배너 + 신상품 / 인기상품(Hot) / 세일상품 섹션 |
| 카테고리 | 아우터, 상의, 원피스, 머플러, 신발, 액세서리, 기타 + New/Sale/Hot 탭 |

| 필터 / 정렬 | 카테고리, 가격대(최소~최대), 정렬(최신순·인기순 등) |
| 상품 상세 | 이미지 갤러리, 사이즈·컬러 옵션, 재고, 할인율, 리뷰, AI 스타일링 CTA |

| 기능 | 설명 |
|------|------|
| 장바구니 | 담기, 수량 변경, 삭제 |

| 리뷰 | 상품별 리뷰 조회, 작성/수정/삭제, 별점 |

## 6. 갤러리 (커뮤니티)

AI 스타일링 결과물을 공유하는 소셜 피드 기능. 경로 `/gallery`

| 기능 | 설명 |
|------|------|
| 피드 | 전체 스타일링 결과 둘러보기 + 인기 갤러리 |
| 게시글 작성 | AI 스타일링 결과 이미지 공유 (연결 상품 정보 함께 노출) |
| 좋아요 | 토글 방식, 좋아요 수 표시 |
| 댓글 | 작성 / 삭제 |
| 상세 보기 | 게시글 상세 조회 및 본인 게시글 삭제 |

## 9. 기술적 특징 (보고서 어필 포인트)

- **모바일 우선 반응형 설계** — PC에서도 모바일 앱 UX 그대로 제공
- **JWT 기반 인증 + 토큰 자동 재발급** — 로그아웃 없는 끊김 없는 이용
- **공통 API 응답 구조 및 에러 처리 표준화** — Axios 인터셉터 기반
- **페이지네이션 공통 처리**, 디바운스 검색 등 성능 최적화
- **컴포넌트 재사용 설계** 및 **TypeScript 타입 안정성** 전반 적용
- **파일 업로드** — 반려견 사진, 스타일링 결과 이미지 등 (S3 연동)

## 10. 참고: 시연 시 확인 필요 사항

- 현재 AI 스타일링 합성은 **Mock(샘플) 모드**로 동작 중
  - `src/services/aiStylingService.ts`의 `USE_MOCK = true` 설정
  - Mock 모드에서는 실제 AI 서버 대신 약 2초 대기 후 샘플 결과 이미지 반환
  - 실제 합성 동작을 위해서는 해당 값을 `false`로 변경 필요
- 보고서는 "백엔드 연동 완료" 기준으로 작성 가능하나, 실제 시연/심사 시 위 설정 확인 권장
