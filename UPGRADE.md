# PetFit 서비스 고도화 및 비즈니스 확장

우리 서비스의 핵심 가치인 **'AI 가상 피팅'**을 기반으로, 사용자 편의성을 향상시키고 지속 가능한 수익 모델을 구축하기 위한 향후 개발 계획을 다음과 같이 정의한다.

## 1순위: 데이터 기반 사이즈 추천 AI 고도화

단순 시각화를 넘어 데이터 기반의 정확한 사이즈 가이드 제공

### 배경

현재 AI 스타일링은 이미지 기반으로 동작하여 견종별 체형 차이(흉통, 등 길이 등)를 충분히 반영 하지 못하는 한계가 존재한다.

### 주요 개발 내용

반려견 상세 프로필 시스템 구축

- 견종(Breed), 나이(Age), 체중(Weight)
- 목 둘레(Neck), 가슴 둘레(Chest), 등 길이(Back Length)
- 스키마 설계 및 DB 마이그레이션 진행
  AI 프롬프트 엔지니어링 고도화
- 스타일링 생성 시 이미지 + 프로필 데이터를 함께 입력
- 체형에 최적화된 결과 생성
  유사 체형 데이터 큐레이션
- 유사 체형 반려견 그룹화
- 구매 및 리뷰 데이터 기반 추천 제공
- 예: “이 강아지와 체형이 비슷한 사용자 78%가 L 사이즈를 선택했습니다.”

### 기대 효과

- 구매 실패율(반품률) 감소
- 사이즈 추천 정확도 향상
- 사용자 신뢰도 확보
