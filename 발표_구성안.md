# PetFit — PPT 슬라이드 구성안 (7분)

> **총 14장** | 슬라이드당 평균 30초
> 각 슬라이드는 **[화면 내용]** (슬라이드에 표시) + **[발표자 노트]** (말할 내용)로 구성

---

## SLIDE 01 — 타이틀
⏱ 진입 시 표시 (별도 시간 없음)

### 화면 내용
```
🐾 PetFit

AI 기반 반려동물 의류 이커머스 플랫폼

[팀명 / 발표자 이름 / 날짜]
```

### 발표자 노트
> 청중이 착석하는 동안 표시. 발표 시작 전 잠깐 대기.

## SLIDE 03 — 서비스 배경
⏱ 약 15초

### 화면 내용
```
강아지 옷 구매 후 반품,
다들 한 번씩은 있으실 겁니다.

저희 팀은 이 불편함에서 출발했습니다.
```
*(텍스트만, 큰 폰트. 배경은 여백 많게)*

### 발표자 노트
> "먼저 저희 서비스가 시작하게 된 이유입니다.
> 많은 사람들이 강아지 옷을 샀다가 사이즈가 안 맞아서 반품을 많이 했습니다.
> 저희 팀은 이 단순한 불편함에서 출발했습니다."

## SLIDE 04 — 문제 정의
⏱ 약 30초

### 화면 내용
```
왜 반품이 생길까?

😣  견종마다 체형이 다르다
     (같은 몸무게여도 흉통·등 길이가 전혀 다름)

📏  사이즈 가이드만으로는 판단 불가

🚫  구매 전 입혀볼 수 없다

     → 높은 반품률 = 소비자 불만 + 셀러 손실
```

### 발표자 노트
> "문제의 핵심은 세 가지입니다.
> 첫째, 견종마다 체형이 전혀 다릅니다. 같은 3kg이어도 말티즈와 닥스훈트의 등 길이는 다르죠.
> 둘째, 기존 사이즈 가이드는 수치만 제공할 뿐, 실제 핏을 보장하지 않습니다.
> 셋째, 사람과 달리 반려동물은 직접 입혀보기 어렵습니다.
> 결과적으로 반품률이 높아지고, 소비자와 판매자 모두 손해를 봅니다."

## SLIDE 06 — 시장 배경 & 차별점
⏱ 약 30초

### 화면 내용
```
왜 지금, 왜 PetFit인가

🏠  국내 반려동물 양육 가구  약 600만+
📈  반려동물 용품 시장  연평균 고성장 지속

기존 반려동물 쇼핑몰          PetFit
──────────────────────────────────────
일반 이커머스 구조       →   AI 가상 피팅
사이즈 표만 제공         →   체형 데이터 추천
단순 구매               →   커뮤니티 + 공유
```

### 발표자 노트
> "시장 배경을 말씀드리겠습니다.
> 국내 반려동물 양육 가구는 현재 600만 가구를 넘어섰고, 시장은 계속 성장 중입니다.
> 그러나 기존 반려동물 쇼핑몰은 일반 이커머스와 구조가 동일합니다.
> PetFit은 AI 피팅, 체형 기반 추천, 커뮤니티 기능으로 차별화합니다."

## SLIDE 08 — 데모 (Live)
⏱ 약 1분 50초

### 화면 내용
*(앱 화면 직접 시연 — 이 슬라이드는 최소화하거나 숨김)*
```
[ 라이브 데모 ]
```

### 발표자 노트
> **① 상품 탐색 (15초)**
> "먼저 홈 화면입니다. 카테고리별로 상품을 탐색할 수 있고, 가격 필터와 정렬 기능도 있습니다."
>
> **② 맞춤 사이즈 추천 (20초)**
> "상품 상세 페이지에 들어오면, 등록된 반려견 프로필을 선택해 맞춤 사이즈를 추천받을 수 있습니다. 단순 수치 비교가 아니라, 체형이 비슷한 실제 사용자 구매 데이터를 기반으로 추천해드립니다."
>
> **③ 반려견 프로필 (20초)**
> "마이페이지에서 반려견 프로필을 등록합니다. 이름, 견종, 체중과 함께 가슴둘레, 등 길이 같은 체형 정보를 입력하는데, 이 데이터가 AI 정확도와 사이즈 추천 정확도를 높이는 핵심입니다."
>
> **④ AI 스타일링 (40초)**
> "이제 핵심 기능입니다. AI 스타일링 탭에서 반려견 사진을 업로드하고, 상품을 선택합니다. 사이즈와 컬러도 이 단계에서 고를 수 있어요. 생성 버튼을 누르면 AI가 가상 피팅 이미지를 만들어줍니다. 구매 전에 어떻게 보이는지 미리 확인할 수 있는 거죠."
>
> **⑤ 피드 공유 (20초)**
> "결과 이미지는 커뮤니티 피드에 바로 공유할 수 있습니다. 다른 반려인들의 스타일링도 함께 볼 수 있고, 좋아요와 댓글로 소통할 수 있어요."
>
> **⑥ 장바구니 → 주문 (10초)**
> "마음에 드는 상품은 장바구니에 담아 바로 주문할 수 있습니다."
>
> **⑦ 리뷰 작성 (15초)**
> "배송이 완료된 상품은 마이페이지 주문 내역에서 리뷰를 작성할 수 있습니다. 실제 구매자의 리뷰가 쌓일수록 다른 반려인들의 사이즈 선택에도 도움이 됩니다."

## SLIDE 09 — AI 스타일링 시스템
⏱ 약 30초

### 화면 내용
```
AI 스타일링 — 어떻게 작동하나요?

반려견 사진          의류 사진
    +         +    반려견 프로필 데이터
    ↓
  AI 모델 (이미지 합성)
    ↓
  가상 피팅 결과 이미지
  (S3 저장 + 이력 관리)
```

### 발표자 노트
> "AI 스타일링의 작동 방식을 설명드리겠습니다.
> 반려견 사진과 의류 사진만이 아니라, 반려견의 견종·체형 프로필 데이터를 함께 AI 모델에 주입합니다.
> 단순한 이미지 합성이 아니라, 체형 정보가 반영된 더 정확한 결과를 만들어내는 거죠.
> 생성된 이미지는 AWS S3에 저장되어 언제든 다시 확인할 수 있습니다."

## SLIDE 10 — 체형 기반 사이즈 추천
⏱ 약 30초

### 화면 내용
```
사이즈 추천 — 데이터가 말해줍니다

반려견 가슴둘레 기준 ±20% 유사 체형 사용자 분석

  "이 강아지와 체형이 비슷한 사용자의
   78%가 L 사이즈를 선택했습니다"

사이즈    핏 평가       설명
─────────────────────────────
  S    너무 작음    권장하지 않음
  M    최  적      가장 잘 맞습니다  ✅
  L    여유 있음    약간 여유가 있어요
```

### 발표자 노트
> "두 번째 핵심 기능은 데이터 기반 사이즈 추천입니다.
> 단순히 수치를 비교하는 것이 아니라, 가슴둘레 기준 유사한 체형의 실제 사용자 구매 데이터를 분석합니다.
> '비슷한 체형의 사용자 78%가 L을 선택했다'는 식으로 근거 있는 추천을 제공해 신뢰도를 높입니다.
> 이 데이터가 쌓일수록 추천 정확도는 더 높아집니다."

## SLIDE 11 — 기술 스택
⏱ 약 30초

### 화면 내용
```
기술 스택

프론트엔드              백엔드                인프라
──────────────────────────────────────────────
React + TypeScript  →  Spring Boot (Java)  →  AWS S3
Tailwind CSS            REST API               JWT 인증
Zustand                 JPA / ORM              AI 모델 연동
```

### 발표자 노트

> 프론트엔드는 React와 TypeScript로, 백엔드는 Spring Boot로 구성했습니다.
> 이미지는 AWS S3에 저장하고, JWT 인터셉터로 토큰 자동 갱신을 처리해 보안과 UX를 모두 잡았습니다."
